## [0.1.0] - 2024

### Added

- Initial release of Lele QDA application
- Project management with SQLite persistence
- Source import for multiple formats (text, PDF, Word, audio, video, images)
- Hierarchical coding nodes with color coding
- Full-text search using SQLite FTS5
- Auto-coding with sentence-transformers and HDBSCAN clustering
- Visualization tools (word clouds, charts, mind maps, sociograms)
- Audio/video transcription using OpenAI Whisper
- French localized interface
